pastikan sudah install git cli

pastikan branch git di master cek dengan git branch -a

bersihkan kalo ada remote yang kemarin kemarin (cek git config --list dan git branch -a)

pastikan sudah install heroku cli

cd ke folder project

ketik pip install gunicorn

buat daftar library yang di butuhkan project kita
ketik pip freeze > requirements.txt

buat file bernama Procfile tanpa exktensi
isi web: gunicorn app:app saja

ketik heroku login

buat project di heroku.com

ketik git init .

ketik git add --all                   (setiap ada perubahan mau push mulai dari sini)

setup username dan email git
git config --global user.email "emailgithub"
git config --global user.name "<usernamegithub>"

ketik git commit -m "update"

connectkan git dan heroku
ketik heroku git:remote -a <namaprojectheroku>

eksekusi file ke heroku
ketik git push heroku master
